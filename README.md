# 逻辑电路模拟系统

## 一、运行环境

**后端环境**
- 操作系统：Windows 10/11、macOS、Linux、HarmonyOS
- Python版本：3.6 或更高
- 依赖包：Flask 3.0+

**前端环境**
- 浏览器：Google Chrome、Mozilla Firefox、Microsoft Edge 等现代浏览器
- 网络连接：需要网络连接以与后端服务器交互

## 二、安装步骤

1. **克隆项目**
   - 下载项目文件到本地目录

2. **安装依赖**
   - 打开命令提示符或PowerShell
   - 导航到项目目录：`cd turing-complete`
   - 运行命令：`python -m pip install flask`

3. **启动服务器**
   - 在项目目录中运行：`python app.py`
   - 服务器将在 http://localhost:5000 启动

## 三、具体操作

**基本操作**

1. **添加元件**
   - 点击工具栏上的元件按钮
   - 或使用数字键快捷键（3-7）

2. **连接元件**
   - 点击元件的输入/输出端口
   - 拖动到另一个元件的端口释放

3. **删除元件**
   - 右键点击元件
   - 或使用 Ctrl+D 删除选中的元件

4. **删除导线**
   - 右键点击导线

5. **切换输入状态**
   - 使用输入切换工具（数字键 2）点击输入元件

6. **缩放视图**
   - 使用鼠标滚轮

7. **移动屏幕**
   - 按住左键拖动空白处

8. **复制元件**
   - 中键点击元件
   - 或 Ctrl+C 复制选中的元件

9. **粘贴元件**
   - Ctrl+V 粘贴复制的元件

10. **全选元件**
    - Ctrl+A 选择所有元件

11. **保存为函数**
    - 选中包含输入输出的模块后，按 Ctrl+S
    - 在弹窗中输入函数名称并保存

12. **使用函数**
    - 点击右侧函数面板中的函数名称放置函数元件

13. **撤销操作**
    - 按 Ctrl+Z 或点击撤销按钮

14. **重做操作**
    - 按 Ctrl+Y 或点击重做按钮

## 四、使用说明

**快捷键**

| 快捷键 | 功能 |
|--------|------|
| 1 | 选择工具 |
| 2 | 输入切换工具 |
| 3 | 添加与门 |
| 4 | 添加或门 |
| 5 | 添加非门 |
| 6 | 添加输入 |
| 7 | 添加输出 |
| 8 | 删除工具 |
| 9 | 清空电路 |
| 0 | 切换网格 |
| Ctrl+Z | 撤销 |
| Ctrl+Y | 重做 |
| Ctrl+C | 复制选中的元件 |
| Ctrl+V | 粘贴元件 |
| Ctrl+A | 全选 |
| Ctrl+S | 将选中电路保存为函数 |
| Ctrl+D | 删除选中的元件 |
| Esc | 取消当前操作 |

**电路状态显示**

- 绿色：表示信号状态为 1
- 红色：表示信号状态为 0
- 灰色：表示导线未连接

**示例电路**

1. **简单的与门电路**
   - 添加两个输入元件
   - 添加一个与门
   - 添加一个输出元件
   - 连接输入元件到与门，与门到输出元件
   - 切换输入状态，观察输出变化

2. **非门电路**
   - 添加一个输入元件
   - 添加一个非门
   - 添加一个输出元件
   - 连接输入元件到非门，非门到输出元件
   - 切换输入状态，观察输出变化

3. **函数保存与复用**
   - 创建包含输入元件和输出元件的电路
   - 使用框选或 Ctrl+A 全选需要保存的元件
   - 按 Ctrl+S，在弹窗中输入函数名称
   - 点击保存，函数会出现在右侧面板
   - 点击函数名称，即可在画布上放置该函数元件
   - 支持多层嵌套：一个函数内部可以包含另一个函数

## 五、注意事项

- 确保所有输入端口都已连接，否则逻辑门可能无法正确计算
- 电路状态会自动保存，刷新页面后会恢复之前的状态
- 对于复杂电路，可能需要更多时间进行计算
- 保存函数时需要选中包含至少一个输入和一个输出的完整电路
- 重复的函数名称会被阻止，请使用不同的名称

## 六、故障排除

1. **服务器启动失败**
   - 检查 Python 是否已正确安装
   - 检查 Flask 是否已正确安装
   - 确保端口 5000 未被占用

2. **电路不工作**
   - 检查所有输入端口是否都已连接
   - 检查导线连接是否正确
   - 尝试刷新页面重新加载电路

3. **保存失败**
   - 检查网络连接
   - 确保服务器正在运行
   - 检查文件权限是否正确
