# 逻辑电路设计与模拟系统

## 项目介绍

这是一个基于 Web 的逻辑电路设计与模拟系统，允许用户通过直观的界面创建、编辑和模拟逻辑电路。系统支持与门、或门、非门、输入和输出元件，并提供实时的电路状态模拟。

## 功能特性

- **直观的拖放界面**：通过鼠标操作轻松创建和连接电路元件
- **实时电路模拟**：元件和导线根据信号状态显示不同颜色（1 为绿色，0 为红色）
- **丰富的元件库**：支持与门、或门、非门、输入和输出元件
- **键盘快捷键**：提供数字键快捷键，方便快速选择工具和元件
- **鼠标滚轮缩放**：支持通过鼠标滚轮缩放电路视图
- **右键删除**：可以通过右键点击删除元件和导线
- **状态保存**：自动保存电路状态到本地存储和服务器
- **撤销/重做**：支持操作的撤销和重做
- **响应式设计**：适配不同屏幕尺寸

## 技术栈

- **前端**：HTML5 Canvas, JavaScript, CSS3
- **后端**：Python, Flask
- **存储**：本地存储 (localStorage) 和 JSON 文件

## 安装与运行

### 前提条件

- Python 3.6 或更高版本
- pip 包管理器

### 安装步骤

1. 克隆项目到本地
   ```bash
   git clone <repository-url>
   cd turing-complete
   ```

2. 安装依赖
   ```bash
   pip install flask
   ```

3. 运行服务器
   ```bash
   python app.py
   ```

4. 在浏览器中访问
   ```
   http://localhost:5000
   ```

## 使用指南

### 基本操作

1. **添加元件**：点击工具栏上的元件按钮或使用相应的数字键快捷键
2. **连接元件**：点击元件的输入/输出端口并拖动到另一个元件的端口
3. **删除元件**：右键点击元件
4. **删除导线**：右键点击导线
5. **切换输入状态**：使用输入切换工具（数字键 2）点击输入元件
6. **缩放视图**：使用鼠标滚轮
7. **移动屏幕**：按住左键拖动空白处
8. **撤销操作**：按 Ctrl+Z 或点击撤销按钮
9. **重做操作**：按 Ctrl+Y 或点击重做按钮

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| 1 | 选择工具 |
| 2 | 输入切换工具 |
| 3 | 添加与门 |
| 4 | 添加或门 |
| 5 | 添加非门 |
| 6 | 添加输入 |
| 7 | 添加输出 |
| 8 | 删除工具 |
| 9 | 清空电路 |
| 0 | 切换网格 |
| Ctrl+Z | 撤销 |
| Ctrl+Y | 重做 |

### 电路状态显示

- **绿色**：表示信号状态为 1
- **红色**：表示信号状态为 0
- **灰色**：表示导线未连接

## 实现原理

1. **前端渲染**：使用 HTML5 Canvas 绘制电路元件和导线
2. **电路计算**：基于元件的输入状态计算输出状态
3. **状态管理**：使用历史记录实现撤销/重做功能
4. **数据存储**：将电路状态保存到本地存储和服务器
5. **响应式设计**：根据屏幕尺寸调整布局

## 示例电路

### 简单的与门电路
1. 添加两个输入元件
2. 添加一个与门
3. 添加一个输出元件
4. 连接输入元件到与门，与门到输出元件
5. 切换输入状态，观察输出变化

### 非门电路
1. 添加一个输入元件
2. 添加一个非门
3. 添加一个输出元件
4. 连接输入元件到非门，非门到输出元件
5. 切换输入状态，观察输出变化

## 注意事项

- 确保所有输入端口都已连接，否则逻辑门可能无法正确计算
- 电路状态会自动保存，刷新页面后会恢复之前的状态
- 对于复杂电路，可能需要更多时间进行计算
